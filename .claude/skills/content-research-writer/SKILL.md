- Writing blog posts and articles
- Creating technical documentation

- Gathering credible sources and citations
- Writing educational content
- Producing research-backed writing
- Creating white papers and reports

- Developing content strategies

## When to use this skill

- Writing technical content with citations.
- When working on related tasks or features

**Use when**: Writing technical content with citations.

2. Extract key points
3. Add citations
4. Review for accuracy
